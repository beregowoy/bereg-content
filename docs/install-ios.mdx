# Установка на iOS

Инструкция подходит для iPhone и iPad с iOS 15 и выше.

## Шаг 1. Установите приложение

Откройте App Store на устройстве и установите один из поддерживаемых клиентов:

- **Happ** — рекомендуемый клиент для iOS.
- **Streisand** — альтернатива, если Happ не работает.
- **V2Box** — ещё одна альтернатива.

Все приложения бесплатны.

## Шаг 2. Получите ссылку для подключения

1. Откройте [личный кабинет](https://app.beregvpn.com) на телефоне.
2. Перейдите в раздел **Мои подписки**.
3. Нажмите **Подключить** → **iOS** и выберите установленное приложение.

Система откроет приложение и автоматически добавит конфигурацию.

<Callout type="info" title="Если приложение не открылось">
  Скопируйте ссылку подписки из кабинета (кнопка «Скопировать ссылку»),
  откройте приложение вручную и вставьте её в поле **Add Subscription**
  или **Import from URL**.
</Callout>

## Шаг 3. Разрешите VPN

При первом подключении iOS покажет запрос **«Разрешить добавить конфигурации VPN»** —
нажмите **Разрешить** и подтвердите действие через Face ID / Touch ID или код-пароль.

## Шаг 4. Подключитесь

Выберите сервер в списке и нажмите большую кнопку подключения. В статус-баре
появится значок **VPN** — всё работает.

## Частые проблемы

- **Значок VPN не появился.** Проверьте, что подписка активна в кабинете
  и что вы разрешили конфигурацию VPN в настройках iOS.
- **Приложение не видит серверы.** Перезапустите приложение и обновите
  подписку в нём (**Update subscription** / «Обновить»).
- **Медленно работает.** Попробуйте другой сервер из списка.

Если проблема осталась — напишите в [поддержку](https://t.me/beregvpn_supportbot).
