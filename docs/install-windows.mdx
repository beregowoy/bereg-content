# Установка на Windows

Инструкция подходит для Windows 10 и Windows 11.

## Шаг 1. Скачайте клиент

Рекомендуем один из клиентов:

- **Hiddify** — простой интерфейс, поддержка подписок.
- **Nekoray** — продвинутый клиент с большими настройками.
- **v2rayN** — классический клиент, стабильно работает.

Скачайте установочный файл с официального сайта выбранного клиента.

## Шаг 2. Установите приложение

1. Запустите скачанный установщик.
2. Подтвердите запуск в Windows SmartScreen (нажмите **Подробнее → Выполнить в любом случае**, если появится).
3. Дождитесь завершения установки и откройте программу.

<Callout type="warning" title="Важно">
  Если антивирус блокирует клиент, добавьте его папку в исключения.
  Установщики подписаны и безопасны, но антивирусы иногда реагируют
  на VPN-утилиты.
</Callout>

## Шаг 3. Импортируйте подписку

1. Откройте [личный кабинет](https://app.beregvpn.com) → **Мои подписки**.
2. Скопируйте ссылку подписки.
3. В клиенте найдите пункт **Subscription** (или «Подписки»).
4. Нажмите **Add** и вставьте ссылку.
5. Нажмите **Update subscription** / «Обновить», чтобы получить список серверов.

## Шаг 4. Подключитесь

1. Выберите сервер в списке.
2. Нажмите **Connect** (или правой кнопкой на иконку в трее → **Включить**).
3. В трее появится значок подключённого VPN.

## Частые проблемы

- **Клиент не запускается.** Проверьте, что установлен `.NET Runtime`
  или `Visual C++ Redistributable` (обычно ставится автоматически).
- **Нет интернета.** Переключите режим на **Rule/TUN** или **System Proxy** в настройках.
- **Сайты открываются медленно.** Выберите другой сервер.

Напишите в [поддержку](https://t.me/beregvpn_supportbot), если что-то не работает.
