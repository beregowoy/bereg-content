# Установка на Android

Инструкция подходит для Android 8 и выше.

## Шаг 1. Установите приложение

Откройте Google Play и установите один из клиентов:

- **v2rayNG** — основной рекомендуемый клиент.
- **Happ** — альтернатива с более простым интерфейсом.
- **Hiddify** — ещё одна альтернатива.

Если Google Play недоступен, скачайте APK с официального сайта клиента.

## Шаг 2. Получите ссылку для подключения

1. Откройте [личный кабинет](https://app.beregvpn.com) на телефоне.
2. Перейдите в раздел **Мои подписки**.
3. Нажмите **Подключить** → **Android** и выберите установленное приложение.

## Шаг 3. Импортируйте подписку

В большинстве клиентов это делается так:

1. Откройте клиент.
2. Нажмите «+» в правом верхнем углу.
3. Выберите **Import from clipboard** или **Add subscription**.
4. Если ссылка не подставилась автоматически — вставьте её из буфера.

## Шаг 4. Подключитесь

1. Выберите нужный сервер в списке.
2. Нажмите кнопку подключения внизу экрана.
3. Android спросит разрешение на создание VPN-соединения — нажмите **OK**.

В шторке уведомлений появится значок **Ключ** — VPN активен.

<Callout type="tip" title="Совет">
  Разрешите приложению работать в фоне и отключите экономию батареи
  для клиента VPN, иначе Android может обрывать соединение.
</Callout>

## Частые проблемы

- **Нет интернета после подключения.** Попробуйте другой сервер.
- **Соединение обрывается.** Снимите ограничения экономии батареи.
- **Ошибка импорта подписки.** Скопируйте ссылку заново из кабинета.

Если не получается подключиться — напишите в [поддержку](https://t.me/beregvpn_supportbot).
