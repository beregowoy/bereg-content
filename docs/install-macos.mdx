# Установка на macOS

Инструкция подходит для macOS 12 Monterey и новее, работает на Intel и Apple Silicon.

## Шаг 1. Установите клиент

Рекомендуемые приложения:

- **Happ** — доступно в Mac App Store, самый простой вариант.
- **V2Box** — также в Mac App Store.
- **Hiddify** — DMG с официального сайта проекта.

Для App Store просто нажмите **Загрузить**, для DMG-файла — перетащите
приложение в папку **Applications** после скачивания.

## Шаг 2. Получите ссылку для подключения

1. Откройте [личный кабинет](https://app.beregvpn.com) → **Мои подписки**.
2. Нажмите **Подключить** → **macOS**.
3. Скопируйте ссылку или нажмите **Открыть в приложении**.

## Шаг 3. Импортируйте подписку

1. Запустите клиент.
2. Откройте раздел **Subscription** / «Подписки».
3. Нажмите **+** или **Add**, вставьте скопированную ссылку.
4. Нажмите **Update** — клиент загрузит список серверов.

## Шаг 4. Подключитесь

1. Выберите сервер.
2. Нажмите большую кнопку **Connect**.
3. macOS спросит разрешение на добавление VPN-конфигурации — введите пароль администратора.

В строке меню появится иконка VPN — соединение работает.

<Callout type="info" title="Если кнопка не активна">
  Перейдите в **Системные настройки → VPN** и проверьте, что добавленный
  профиль не заблокирован. Иногда macOS требует разрешить расширение в
  **Конфиденциальность и безопасность**.
</Callout>

## Частые проблемы

- **Не видно подписки.** Нажмите **Update subscription** в настройках клиента.
- **Нет интернета.** Переключите режим на **TUN/Global** в настройках.
- **Приложение не запускается.** ПКМ по иконке → **Открыть** в первый раз.

Напишите в [поддержку](https://t.me/beregvpn_supportbot), если нужна помощь.
